# Weather App

Веб-приложение для отображения актуальной информации о погоде в любом городе мира с адаптивным дизайном.

## Технологии

- **HTML5** - структура приложения
- **CSS3** - стилизация (Flexbox, Grid, градиенты, медиа-запросы)
- **Vanilla JavaScript** - логика приложения
- **OpenWeather API** - получение данных о погоде
- **SVG** - векторные иконки погоды

## Возможности

- Текущая погода и прогноз на 4 дня
- Поиск погоды по любому городу мира
- Динамическая смена фона в зависимости от погодных условий
- Отображение детальной информации: влажность, осадки, скорость ветра
- Полностью адаптивный дизайн для всех устройств
